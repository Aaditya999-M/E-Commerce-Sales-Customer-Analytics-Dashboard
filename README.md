# E-Commerce-Sales-Customer-Analytics-Dashboard
End - to - End Power BI Dashboard Analyzing Sales, Marketing Performance And Customer Behavior
## Project Overview

This project analyzes e-commerce sales performance, marketing effectiveness, and customer behavior using Power BI. The objective was to transform raw transactional and website session data into actionable business insights.

## Tools Used

- Power BI
- Power Query
- DAX
- Data Modeling
- Excel / CSV

## Dashboard Pages

### Executive Overview
- Revenue Analysis
- Profit Analysis
- Conversion Rate
- Average Order Value

### Marketing Analytics
- Traffic Source Analysis
- Campaign Performance
- Marketing Funnel
- Landing Page Analysis

### Customer Behavior Analytics
- Customer Lifetime Value
- Repeat Purchase Rate
- Purchase Frequency
- Cross-Sell Analysis

## Key Insights

- Acquired over 32K customers.
- Customer Lifetime Value reached $61.16.
- Repeat Purchase Rate was 1.86%.
- Cross-Sell Rate increased from 2.18% to over 25%.
- Google Search was the leading acquisition channel.

## Business Recommendations

- Improve customer retention strategies.
- Expand cross-selling initiatives.
- Optimize landing pages.
- Focus on increasing customer lifetime value.
